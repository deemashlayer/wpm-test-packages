Add OpenID Provider User ID Token Claims
Screen Heading: Security > User Management > Add and Remove OpenID Provider Users > Add and Remove OpenID Provider User Claims > Add OpenID Provider User Id Token Claims
Use the Add OpenID Provider User Id Token Claims screen to add ID Token claims for a particular OpenID Provider user.
To open the Add OpenID Provider User Id Token Claims screen, go to the User Management Screen and click Add and Remove OpenID Provider Users. Under OpenID Provider Users, click Edit in the row for the user for which you want to edit, assign, or remove claims. Click Add ID Token Claim.
Click Return to Add and Remove OpenID Provider Users Claims to return to the list of OpenID Provider users.
