Screen Heading: Packages > Publishing
Use the Publishing Packages screen to view the list of releases currently available to subscribers and to send a release to one or more subscribers.
To open the Publishing Packages screen, click Publishing under the Packages menu on the Navigation panel.
Click Create and Delete Releases to generate a new release of a package or remove an existing one.
Click Add Subscribers to add subscribers to the subscription list for a package.
Click Update and Remove Subscribers to remove subscribers from the subscription list for a package.
Available Releases
Displays the names of the releases (distribution files) that are currently available to subscribers. Releases reside in the Integration Server_directory \instances\instance_name\replicate\outbound directory. To send a release to one or more subscribers, click the Send Release link for that release.
