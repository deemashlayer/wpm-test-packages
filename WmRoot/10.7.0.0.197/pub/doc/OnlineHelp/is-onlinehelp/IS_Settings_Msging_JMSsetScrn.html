JMS Settings Screen
Screen Heading: Settings > Messaging > JMS Settings
Use the JMS Settings screen to view JMS connection alias information.
To open the JMS screen, click Messaging under the Settings menu on the Navigation panel. Under JMS Configuration on the Messaging Screen, click JMS Settings.
Click Return to Messaging to return to the previous screen.
Click View JMS Trigger Management to view global and individual controls for JMS triggers
Click Create JMS Connection Alias to create a JMS connection alias.
Click Refresh Page to update the information on the screen.
JMS Connection Alias Definitions
Connection Alias Name
Name of the JMS connection alias. Each JMS connection alias represents a connection factory to a specific JMS provider.
Description
A description of the JMS connection alias.
Transaction Type
Indicates whether sessions that use this JMS connection alias will be transacted.
Transaction Type
Description
NO_TRANSACTION
Sessions that use this JMS connection alias are not transacted.
LOCAL_TRANSACTION
Sessions that use this JMS connection alias are part of a local transaction.
XA_TRANSACTION
Sessions that use this JMS connection alias are part of an XA transaction.
CSQ Count
Specifies the number of messages in the client side queue (CSQ) for this JMS connection alias.
Enabled
Indicates whether or not the JMS connection alias is enabled.
Delete
Click to delete the JMS connection alias.
