Current Sessions Screen
Screen Heading: Server > Statistics > Sessions
Use the Sessions screen to view detailed information about the active sessions on the webMethods Integration Server.
To open the Current Sessions screen, first go to the Statistics Screen. In the Usage section, find the Total Sessions row or the Stateful Sessions row and click the number link in the Current column.
Click Return to Server Statistics to return to the main statistics screen.
Click Kill All Sessions Except Your Session to kill all sessions except the session you are currently running.
Note:
You will not be able to kill sessions if the masking of Session IDs is enabled.
User
Displays name of the directory and user who initiated session (e.g., Administrator). The icon highlights the session you are currently running.
From
Identifies the source from which the user initiated the session.
Stateful
Indicates whether or not the session is a stateful session. True indicates that the session is stateful.
Requests
Displays number of service invocations that this user has requested during this session.
Connect Time
Displays the amount of time (in seconds) that the session has been active.
Last Request
Displays the amount of time (in seconds) since the last service invocation made during this session.
Session Expires
Displays the amount of time (in seconds) until the session expires if the server doesn't receive another request from this user.
Kill
Cancels the session immediately. To cancel a session, click the icon in this field.
Note:
You will not be able to cancel a particular session if the masking of Session IDs is enabled.
