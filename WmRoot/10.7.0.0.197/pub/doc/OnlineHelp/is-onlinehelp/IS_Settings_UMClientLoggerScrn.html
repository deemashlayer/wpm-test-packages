View UM Client Logger Details Screen
Screen Heading: Settings > Logging > View UM Client Logger Details
Use this screen to view settings for the UM Client logger which writes to the Universal Messaging client log file named umClient.log.
To open the View UM Client Logger Details Screen, click Logging under the Settings menu on the Navigation panel. From the logger list, click UM Client Logger.
Click Edit UM Client Logger to change the settings for the session logger.
UM Client Logger Configuration
Log Level
Level of the information that is written to the log. Each level outputs log entries with that level or higher.
Log Size
Maximum size of the log file. When this size is reached, Integration Server rolls the file over to a backup called umClient(number).log and creates a new file.
Log Depth
Number of backup log files to keep on disk when using log rolling. When this number is reached, Integration Server deletes the oldest log file.
