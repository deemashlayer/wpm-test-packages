Edit Digital Event Services Connection Alias Screen
Screen Heading: Settings > Messaging > webMethods Messaging Settings > Connection Alias > Digital Event Services Connection Alias > Edit
To open the Edit Digital Event Services Connection Alias screen, click Messaging under the Settings menu on the Navigation panel. On the Messaging Screen, click webMethods Messaging Settings.
From the webMethods Messaging Settings Screen, disable the IS_DES_CONNECTION connection alias, and then click the IS_DES_CONNECTION connection alias. From the Universal Messaging Connection Alias Details Screen, click Edit Digital Event Services Connection Alias.
Click Return to Digital Event Services Connection Alias to return to the previous screen.
General Settings
Connection Alias Name
Name of the messaging connection alias, specifically IS_DES_CONNECTION. The alias name cannot be modified.
Description
Description for the IS_DES_CONNECTION messaging connection alias.
Client Prefix
A string that identifies the Integration Server to Universal Messaging. If your Integration Server belongs to a cluster, make sure all servers in the cluster use the same client prefix.
Save Changes
Saves the specified settings.
