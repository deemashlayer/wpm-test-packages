Edit Mobile Applications Screen
Screen Heading: Security > Enterprise Gateway Rules > Mobile Application Protection Options > Mobile Applications > Edit
Use the Edit Mobile Applications screen to specify the names of mobile applications from which you want to restrict requests. Mobile applications listed here are available for you to specify in a mobile application protection filter in an Enterprise Gateway rule.
Note:
This screen is disabled if webMethods API Gateway is installed on Integration Server.
To open the Edit Mobile Applications screen, click Enterprise Gateway Rules under the Security menu on the Navigation panel, click Mobile Application Protection Options, and then click Edit Mobile Applications.
Click Return to Mobile Application Protection Options to return to the Mobile Application Protection Options screen.
Mobile Applications
Mobile applications from which you want to restrict or deny requests. Specify one mobile application per line. The mobile applications you specify here will be available for you to specify in a mobile application protection filter in a rule. For example, you can deny requests from pre-3.0 versions of SmallApp.
